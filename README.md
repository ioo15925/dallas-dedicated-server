**IPMI：为什么这一点值得单独说**

很多人选独立服务器时会忽略 IPMI（Intelligent Platform Management Interface）的重要性，但踩过坑之后就明白了。

IPMI 是一套独立于操作系统的带外管理接口。简单说：你的系统崩了、内核没了、SSH 连不上了，通过 IPMI 仍然能连进机器，看到机器的屏幕输出，执行重启、重装、修复操作。没有 IPMI，这种情况通常只能提工单等机房工作人员手动操作。

GTHost 达拉斯独立服务器**全系标配 IPMI**，不是加费选配，是所有套餐直接包含。对于需要自己折腾系统的用户，这一点省去了非常多麻烦。
